## Project Description

This website is designed for individuals with autism based on Human-Computer Interaction (HCI) principles. The goal is to reduce sensory overload and provide a calm, structured, and accessible online environment.

The interface focuses on clarity, readability, and predictable navigation to improve comfort and usability.

---

## Objectives

* Reduce sensory distractions
* Improve readability
* Provide structured navigation
* Support different interaction methods
* Create a peaceful and user-friendly experience

---

## Features

### Simple and Structured Design

* Clean and uniform layout
* Organized menus
* Minimal visual clutter

### High Readability

* Clear fonts with proper spacing
* Black text on white background or white text on black background
* Theme switching option

### Calm Audio Feedback

* Soft sounds for confirmations
* No loud or sudden noises

### Multiple Interaction Methods

* Mouse support
* Keyboard accessibility
* Voice interaction
* Gesture-based interaction

---

## HCI Principles Applied

* Consistency and predictability
* Clear visual hierarchy
* Reduced cognitive load
* Proper color contrast
* Flexible user control
